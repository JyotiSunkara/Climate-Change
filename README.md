# Climate-Change
Data visualization to find climate change buried in millions of data points obtained from verified source: GHCN(Global Historical Climatology Network)

## Getting Started

Enter the directory containing the code and procced to run the following command:

```
    http-server --cors
```

Navigate to:

```
    http://127.0.0.1:8080
```

And open the following html pages to view each unique visualization!

